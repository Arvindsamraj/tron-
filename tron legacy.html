Games on the Grid

This category contains the points of interest and gladiatorial games that are played on the Game Grid.

Light Cycle Game

The game is played by driving a vehicle known as a light cycle, which leaves an impenetrable barrier called a Jetwall.

The Ring Game

The Ring Game is a game played on the Game Grid, usually by programs in gladiator-style matches. The game is also known as Hyperball.

Disc Arena

The Disc Arena is a location in the Game Grid, where programs play gladiator disc games against other programs. The games usually involve two, four or five players.
